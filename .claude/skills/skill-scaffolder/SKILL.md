# Skill Scaffolder

**Quickly bootstrap well-formed skill directories that follow STRUCTURE.md standard.**

## The Essentials

### 1. Create a Skill Skeleton
- Input: skill name, description, 2-3 use cases, 3-5 procedure tasks
- Output: Complete directory with SKILL.md + procedures/ ready for content
- Time: 2 minutes

### 2. Validate Structure
- Check: Frontmatter, routing table, procedure templates
- Ensure: Matches STRUCTURE.md standard
- Prevents: Entropy and inconsistency
