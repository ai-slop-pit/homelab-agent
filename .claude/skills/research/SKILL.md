# Skill: Gemini CLI — Web Research & External Context

## Overview
Use Gemini CLI for web research and accessing current external context. Gemini 3.5 Flash (May 2026) is optimized for autonomous agents.

## Core Purpose
- Web research
- Current facts and updates
- External context
- Information offload to web-powered LLM
- Model selection based on task needs

## Usage

```bash
# Use latest Gemini 3.5 Flash (optimized for agents)
gemini "<your question>"

# Or specify model explicitly
gemini --model "gemini-3.5-flash" "<question>"
gemini --model "gemini-3.1-pro" "<question>"
```

## Models (May 2026)

| Model | Use Case |
|-------|----------|
| **3.5 Flash** | Agentic tasks, research, general use (LATEST - use this) |
| **3.5 Pro** | Coming June 2026, even more capable |
| **3.1 Pro** | Complex reasoning, analysis, high-stakes decisions |
| **3.1 Flash-Lite** | Speed critical, quick facts, low-latency |
| **3 Flash** | Legacy workhorse model |

## When to Use Gemini
- Need current/recent information
- Looking up external APIs or tools
- Getting perspectives beyond local knowledge
- Need context that's not in project/memory
- Research that requires reasoning

---

**Primary research tool** — Gemini 3.5 Flash is built for agents. Use it organically when needing external context.
